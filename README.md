# PCLR-ECG-Lead-Reduction

## 📌 Overview

This repository contains the code and resources for the paper:

**"PCLR-ECG-Lead-Reduction"**

The project focuses on ECG lead reduction using machine learning techniques, aiming to reduce the number of required leads while preserving diagnostic performance.

---

## 🧠 Method Summary

We apply representation learning and predictive modeling techniques to reconstruct or approximate full-lead ECG signals from a reduced subset of leads.

Key components include:

* Data preprocessing and normalization
* Model training (deep learning-based)
* Evaluation on reconstruction and/or classification performance

---
